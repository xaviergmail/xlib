XLIB.tonumber (str)
Integer-safe tonumber This function will return nil if the number to be converted exceeds the integer-definable range of IEEE 754 doubles

Parameters:

  • str string String to be converted to number

Returns:

    number The converted number, if the conversion is safe

Or

    nil Nil if the conversion would lose integer precision
XLIB.FreezeProp (ent)
Freezes an entity in place This sets the entity's movetype to MOVETYPE_NONE and disables motion on its physics object in order to avoid clientside prediction errors.

Parameters:

  • ent Entity Entity to freeze
XLIB.ColorToInt (color)
Serializes a Color object into an integer This is a useful way to store a Color object in binary format as it only uses one double, opposed to 4 (Or god forbid a String)

Parameters:

  • color Color Color to convert to an int

Returns:

    number The color object represented as an integer
XLIB.IntToColor (int)
Deserializes an integer into a Color object

Parameters:

  • int number Integer to convert to a Color object

Returns:

    Color

See also:

    XLIB.ColorToInt
XLIB.SafeColor (color, noalpha)
Clamps all values of a color object between 0-255

Warning:

    This modifies the color object passed to it rather than creating a copy!

Parameters:

  • color Color The color object to clamp
  • noalpha bool[opt=false] Forces the alpha channel to 255 if true

Returns:

    Color The color object passed as the first argument
XLIB.SafeColorNoAlpha (color)
Shortcut for XLIB.SafeColor(..., true) Clamps all the values of a color object between 0-255 and forces the alpha to 255

Warning:

    This modifies the color object passed to it rather than creating a copy!

Parameters:

  • color Color The color object to clamp

See also:

    XLIB.SafeColor
